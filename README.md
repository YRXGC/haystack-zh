# 欢迎来到haystack 
> [官方文档地址](http://django-haystack.readthedocs.org/en/v2.4.1/index.html)

Haystack为Django提供了模块化的搜索。它的特点是统一的，熟悉的API，可以让你在不修改代码的情况下使用不同的搜索后端（比如 Solr, Elasticsearch, Whoosh, Xapian 等等）
